Hex installer (SAMSUNG)

Like substratum, but with less individuality.

Yes, it is a paid app. As for APKs, i have only found one working APK. Said APK does not support plugin themes.
I will NOT post it.

What does it do

It allows you to create custom galaxy store themes for yourself.
You enter some colors, set the plugin, mess with settings if you want, and compile.

My setup

Personally, I use a gray pixel-based interface. Needs AOSP Q plugin (free)

Download ZIP

How to:

  1. put hex in the "Hex\projects" directory
  2. use image as wallpaper
  3. put the font in the "Hex\fonts" directory
  4. Make sure to include font.
I used nova launcher and pixel icon pack. Better yet, check it out in the "shared" portion of the #hexified app, named "AOSP Q blue".
