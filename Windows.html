UWP Explorer

A new file manager hidden in windows.

How to enable

  1. Find a local folder in your computer for the shortcut
  2. Create a shortcut with the file path "explorer shell:AppsFolder\c5e2524a-ea46-4f67-841f-6a9465d9d515_cw5n1h2txyewy!App"
  3. Name it whatever you want
  4. click on it.
